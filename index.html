Problem

데이터는 쌓였지만,
결정은 여전히 감에 의존합니다

이 간극이 부상을 만들고, 선발 논쟁을 만들고, 연맹 보고를 어렵게 만듭니다.

"데이터는 있다. 해석은 사람마다 다르다. 결정은 경험에 의존한다."
이것이 현재 스포츠 현장의 구조입니다.
Daily Ice Decision OS는 이 구조를 바꿉니다

Architecture

제품 구조 — 3 Layer

이 구조는 데이터를 '결정'으로 바꾸기 위해 설계되었습니다
Layer 01
Data Layer
현장의 모든 측정 데이터를 통합 수집. 보이지 않는 상태를 수치로 만듭니다.
  • K-Grip API 실시간 (Kinvent)
  • 오프아이스 측정 — 연 2회
  • Add/Abd · Nordic · RSA · GLSI
  • PHV 성숙도 자동 보정
핵심 엔진 — 우리의 차별점
Layer 02
Decision Engine
데이터를 해석하고 결정으로 연결합니다. 기존 시스템이 분석까지라면, 우리는 결정까지입니다.
  • 위험 분류 (Critical / High / Medium)
  • 포지션별 판단 로직 (GK 전용)
  • 훈련 자동 수정 권고
  • 복귀 타임라인 + 기준값
Layer 03
Output Layer
결정을 전달하고 기록합니다. 이 데이터가 쌓일수록 결정은 더 정확해집니다.
  • 감독/코칭스태프 대시보드 (7화면)
  • 연맹 관리자 뷰 (2화면)
  • 공식 보고서 자동 생성
  • A4 개인 결과 카드 PDF
기존 시스템은 '분석'까지 — 우리는 '결정'까지

Core Value

3가지 경쟁 우위

⚖️
Decision Standardization
사람마다 달랐던 결정을 표준화합니다. 어떤 코치가 보더라도 동일한 데이터 기반으로 동일한 판단 구조를 갖습니다. 논쟁이 없어집니다.
🔍
Explainability (XAI)
"왜 이 결정인가"를 항상 설명할 수 있습니다. 측정 수치 → 위험 판단 → 경기 영향 → 코치 결정까지 전 과정이 투명하게 추적됩니다.
🛡️
Accountability
모든 결정에 데이터 근거가 기록됩니다. 부상 발생 시 사전 경고 여부, 조치 내역, 복귀 기준이 모두 추적 가능합니다. 책임이 구조화됩니다.
